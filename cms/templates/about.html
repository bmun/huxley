About the Creators

CMS was created by Kunal Mehta and William Chieng, two 3rd-year Cal Computer Science undergraduates. Their hobbies including writing code and kicking ass.

Our Tools

We used a lot of great tools to put this app together. Check them out below:

  • Django - An open-source web development framework written in Python that makes building web applications simple.
  • jQuery - An API built on top of JavaScript that makes writing rich, interactive, cross-browser web pages a breeze. "Write less, do more!"
  • Python - An interactive programming language that stresses readability and ease of use. And having a more awesome name than any language out there.
  • Subversion - A version-control system designed to save us from ourselves. SVN makes it easy to keep track of your changes and revert back to an older version when you (inevitably) screw up.
  • Apache - Faithfully serves the app to you every time you ask, without asking a thing in return. The most noble web server there is!
  • Nginx - Serves static content at blazing speeds! You won't know what hit you when the images and styles explode out of the screen.

Special Thanks

We want to give a shout-out to the following resources, which were incredibly helpful and without which we may not have been able to complete CMS.

  • Stack Overflow - The ultimate programming question/answer forum, created for programmers by programmers.
  • Validation Plugin - A simple clientside form validation solution that offers both simplicity and flexibility.
  • Hash Change Plugin - So your browser's back button isn't stuck on the same page. ;]
  • Sreekanth Narayanan's Blog - For ideas on how to auto-format phone numbers. Our script is based loosely on his plugin.
